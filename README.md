# 📱 To-Do Reminder App

An Android application to manage daily tasks with reminder notifications and alarm functionality.

---

## 🚀 Features

* ✅ Add Tasks
* ❌ Delete Tasks
* ✏️ Edit Tasks
* ⏰ Alarm & Notification Reminder
* 🔊 Alarm Sound using MediaPlayer
* 💾 Local Storage using SQLite Database
* 📋 Task List using RecyclerView
* 🎬 Splash Screen

---

## 🛠️ Tech Stack

* Kotlin
* Android SDK
* SQLite Database
* RecyclerView
* BroadcastReceiver
* Foreground Service
* MediaPlayer

---

## 📂 Project Structure

* Activities → MainActivity, AddTaskActivity, SplashActivity
* Adapter → TaskAdapter
* Database → DatabaseHelper
* Receiver → AlarmReceiver, StopReceiver
* Service → AlarmService

---

## ✨ Additional Features

* ✏️ Edit Task Functionality
* 🖱️ Click on task to update
* 🔄 Real-time UI update
